## 토스 앱인토스 웹뷰 개발 환경

### 필수 구성 요소

| 항목 | 내용 |
|------|------|
| **빌드 도구** | Vite (React + TypeScript) |
| **필수 패키지** | @apps-in-toss/web-framework |
| **디자인 시스템** | TDS (토스 디자인 시스템) - 비게임 앱 필수 |
| **설정 파일** | granite.config.ts |

### granite.config.ts 주요 설정

```typescript
{
  appName: string;        // 앱인토스 콘솔의 앱 이름과 동일 (고유 키)
  displayName: string;    // 화면에 표시될 한글 이름
  primaryColor: string;   // TDS 컴포넌트용 RGB HEX 색상값
  scripts: {
    dev: string;         // 개발 서버 실행 명령어
    build: string;       // 프로덕션 빌드 명령어
  };
  outdir: string;        // 빌드 결과물 경로
}
```

### 개발 환경 특성

| 항목 | 설명 |
|------|------|
| **로컬 개발** | HMR 지원, 8081 포트 사용 (샌드박스 전용) |
| **실기기 테스트** | iOS: 동일 와이파이 필요, Android: adb 포트 포워딩 |
| **TDS 제약** | 로컬 브라우저에서 미동작, 샌드박스앱 테스트 필수 |
| **앱 접근** | 개발: intoss://{appName}, 출시: intoss-private://{appName} |
