## 레이어별 상세

### 프론트엔드 (Client)

| 클라이언트 | 기술 | 설명 |
|-----------|------|------|
| **토스 앱인토스 웹뷰** | Vite + React + TypeScript + TDS | 토스 앱 내 웹뷰 서비스 |
| **필수 패키지** | @apps-in-toss/web-framework | 앱인토스 SDK |
| **디자인 시스템** | TDS (토스 디자인 시스템) | 비게임 앱 필수 사용 |
| **약사 전용 웹** | React 또는 Vue.js | 푸시 알림 발송용 관리자 웹 |

### 백엔드 마이크로서비스

| 서비스 | 역할 |
|--------|------|
| **API Gateway** | 요청 수신 → 인증/인가 → 마이크로서비스 라우팅 |
| **사용자/약국 서비스** | 환자·약국 회원 관리, 프로필 |
| **비전/OCR 서비스** | 이미지 수신 → GLM-5 Vision + CLOVA OCR 호출 |
| **LLM 가이드 서비스** | RAG 파이프라인, 가이드 생성, 챗봇 답변 |
| **알림 서비스** | FCM/APNS Push Notification 발송 |

### LLM 가이드 서비스 상세 (핵심)

```
사용자 질문 / 스캔 데이터
        ↓
  Embedding 모델
        ↓
Vector DB 유사도 검색 (약학정보원, KIMS 등)
        ↓
  관련 문서 Retrieval
        ↓
Contextual Prompt 구성
(건강 프로필 + 약물 정보 + 검색 결과)
        ↓
    GLM-5 LLM
        ↓
  맞춤형 복약 가이드 생성
        ↓
TTS 변환 + 실버 모드 UI 출력
```

### 인프라

| 항목 | 기술 선택지 |
|------|-------------|
| **Cloud** | AWS (1순위) 또는 GCP |
| **주요 AWS 서비스** | S3 (이미지 저장), EC2/Lambda (서버), RDS (DB) |
| **정형 DB** | PostgreSQL |
| **벡터 DB** | Pinecone (관리형) 또는 Milvus (자체 호스팅) |
| **메시지 큐** | SQS 또는 Kafka (비동기 처리) |

## 데이터 흐름

```
이미지 (앱 촬영)
  → 암호화 전송 (SSL/TLS)
  → S3 저장 (AES-256 암호화)
  → 비전/OCR 서비스 호출
  → 텍스트 추출
  → LLM 가이드 서비스 (RAG)
  → 가이드 생성
  → 앱 화면 출력 + TTS 재생
  → (약사 발송 시) 푸시 알림 → 환자 앱
```
