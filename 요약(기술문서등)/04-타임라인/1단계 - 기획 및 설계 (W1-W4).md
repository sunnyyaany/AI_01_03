## 리스크

> [!warning]
> - TDS 학습 곡선으로 인한 초기 개발 속도 저하 가능성
> - 웹뷰 환경 제약사항 (카메라, TTS) 대응 필요
> - 3.5주 압축 일정으로 인한 기능 축소 가능성
> - 법률 자문 결과에 따라 기능 범위 조정 가능성

## 웹뷰 전환 주요 변경사항

| 항목 | 기존 | 변경 후 |
|------|------|---------|
| 플랫폼 | 네이티브 앱 (React Native/Flutter) | 토스 앱인토스 웹뷰 |
| 프론트엔드 | React Native/Flutter | Vite + React + TypeScript |
| 디자인 시스템 | 자체 디자인 | TDS (토스 디자인 시스템) 필수 |
| LLM | z.ai GLM-5 | OpenAI GPT-4o |
| 알림 | FCM/APNS | 토스 앱인토스 알림 API |
| 카메라 | 네이티브 API | HTML5 getUserMedia + 파일 업로드 |
| 개발 기간 | 6개월 | 3.5주 (MVP) |
