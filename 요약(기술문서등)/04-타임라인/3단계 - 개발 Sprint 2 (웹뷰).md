## 주요 기술 스택

- **빌드 도구**: Vite
- **프레임워크**: React
- **언어**: TypeScript
- **디자인 시스템**: TDS (토스 디자인 시스템)
- **앱인토스 SDK**: @apps-in-toss/web-framework
- **API 통신**: Axios / Fetch
- **상태 관리**: React Context / Zustand (필요 시)
- **카메라**: HTML5 getUserMedia API
- **TTS**: Web Speech API
- **알림**: 토스 앱인토스 알림 API

## 리스크 및 대응

> [!warning]
> - **TDS 학습 곡선**: 공식 문서 및 예제 코드 참고, 샌드박스 테스트 필수
> - **웹뷰 카메라 제약**: getUserMedia 미지원 시 파일 업로드로 대체
> - **TTS 자동재생 정책**: 사용자 인터랙션 후 재생 트리거 방식 적용
> - **실기기 테스트**: iOS는 동일 와이파이, Android는 adb 포트 포워딩 필요
