## 주요 테스트 시나리오

### 시나리오 1: 처방전 촬영 → 복약 가이드 수신
1. 홈 화면에서 "처방전 촬영" 선택
2. 카메라 또는 파일 업로드로 처방전 이미지 선택
3. OCR 결과 확인 및 수정
4. 복약 가이드 자동 생성 (5초 이내)
5. TTS 자동 재생 확인
6. 알림 등록 확인

### 시나리오 2: 알약 식별
1. 홈 화면에서 "알약 촬영" 선택
2. 알약 이미지 촬영 또는 업로드
3. Vision AI 분석 결과 표시 (5초 이내)
4. 약품 정보 확인

### 시나리오 3: 실버 모드 + TTS
1. 설정에서 실버 모드 활성화
2. 복약 가이드 화면 진입
3. 큰 글씨, 간결한 정보 표시 확인
4. TTS 자동 재생 확인
5. "다시 듣기" 버튼 동작 확인

### 시나리오 4: AI 챗봇
1. 챗봇 화면 진입
2. "이 약 먹고 커피 마셔도 되나요?" 질문
3. RAG 기반 정확한 답변 확인 (5초 이내)
4. 면책 조항 표시 확인
